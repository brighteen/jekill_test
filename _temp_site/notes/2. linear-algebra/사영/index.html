사영 또는 투영은 어떤 집합을 부분집합으로 특정한 조건을 만족시키면서 옮기는 작용이다.

선형대수학에서 말하는 일반적인 사영은 정사영(Orthogonal Projection)을 의미하는데, 이는 바닥으로 수직으로 떨어지는 것을 말한다. 그럼 왜 수직으로 떨어지는 것을 다루는가?
공중에 떠 있는 점에서 바닥까지 갈 수 있는 경로 중 가장 짧은 거리가 수직으로 내려갔을 경우이다. 이를 수학적으로 오차를 최소화한다고 표현한다. 우리가 다루기 힘든 복잡한 문제(고차원 데이터)가 있을 때, 이를 우리가 다룰 수 있는 저차원으로 끌어내릴 때 정보 손실을 가장 적게 하려면 반드시 사영(정사영)이 필요하다.

사영행렬은 멱등성을 가진다. 멱등성은 수학이나 전산학에서 연산의 한 성질을 나타내는 것으로, 연산을 여러 번 적용하더라도 결과가 달라지지 않는 성질을 의미한다(어떤 사영행렬 $P$에 대해 $P^2 = P$).
사영행렬 $P = \begin{bmatrix} 1 & 0 & 0 \ 0 & 1 & 0 \ 0 & 0 & 0 \end{bmatrix}$기 있다. 3차원 공간의 어떤 점에 대해 변환 $P$를 작용시키면 2차원 평면으로 사영된다. $P$를 두 번 곱해보자($P \times P$). 결과는 똑같이 $P$가 나온다.

사영행렬 $P$는 역행렬이 존재하지 않는다. 이전 예시를 다시 들어보자. 3차원 공간의 어떤 점을 2차원 평면으로 사영시킨 점이 있다. 이 점을 다시 3차원 공간의 원래 위치로 복원할 수 있는지를 생각해보면 자명하다. 3차원 공간의 두 벡터 $\mathbf{u}, \mathbf{v}$는 $x_1, x_2$성분이 같지만 $x_3$성분이 다르다. 이때 사영행렬을 두 벡터에 작용시키면 2차원 공간의 같은 지점으로 사영된다. 해당 지점에서 원래 두 벡터로 돌아갈 수 없다.

사영행렬이 역행렬을 가지는 경우는 항등행렬인 경우이다. 항등행렬은 자기 자신을 자기 자신으로 사영시키는 특수한 사영의 형태이다.

Q.그럼 차원을 낮추는 모든 변환은 사영인가?
사영행렬은 멱등성을 가지는데 다음과 같은 행렬 $\begin{bmatrix} 2 & 0 \ 0 & 0 \end{bmatrix}$은 2차원 벡터를 1차원으로 보내며 $x_1$의 크기를 2배 늘리는 변환이다. 이 변환은 멱등성을 가지지 않는다. 행렬 $\begin{bmatrix} 0 & 1 \ 0 & 0 \end{bmatrix}$에 $\begin{bmatrix} x \ y \end{bmatrix}^T$을 곱하면 $\begin{bmatrix} y \ 0 \end{bmatrix}$이 나온다. 이는 차원이 낮아지는 회전변환이며 마찬가지로 멱등성을 가지지 않는다.

Q.차원을 낮추는 모든 변환은 역행렬을 가지지 않는가?
이 변환 또는 함수는 전사함수이다. 하나의 출력에 여러개의 입력이 대응되기 때문에 변환 후 다시 역변환이 불가능하다. 역행렬을 가질 수 없다. 차원을 낮추는 모든 행렬은 가로로 긴 행렬인데 이 직사각형 행렬은 행렬식을 정의할 수 없으므로 역행렬 자체를 정의할 수 없다.
선형대수적 관점으로 보자. 가로로 긴 행렬은 변수의 개수가 방정식의 개수보다 많다. 이 시스템이 해를 가지는 경우 반드시 자유 변수가 존재한다. 즉, $A\mathbf{x}=\mathbf{0}$을 만족하는 $\mathbf{0}$이 아닌 $\mathbf{x}$가 존재하며, 각각의 $\mathbf{x}$에 대응되는 상에 대해서 역변환을 구할 수 없다.
공학에서 완벽한 복구(역변환)대신 근사치를 구하는 의사역행렬(pseudo-inverse)의 개념도 있다.
